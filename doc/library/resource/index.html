More examples on how to use the SAGA resource module can be found in the Resource Module section of the Code Examples chapter.

Like all SAGA modules, the resource module relies on middleware adaptors to provide bindings to a specific resource manager. Adaptors are implicitly selected via the scheme part of the URL, e.g., local:// in the example above selects the local resource adaptor.

Note

A list of available adaptors and supported resource managers can be found in the Developer Documentation part of this documentation.
